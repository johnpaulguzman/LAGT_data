Usage for input_parser.py: 
1. Download Python 3: https://www.python.org/ftp/python/3.6.4/python-3.6.4.exe
2. Within the installation, tick the Check box for: Add or Include Python to System Path
3. Open cmd and cd to this folder
4. Run: python input_parser.py <csv_file> <known_threshold>
- Example: python dataset.csv 60
